Program explanation: Search form for Google

Project chapter 2

This was a first project for CS50 web programing course, the idea was write a front-end for google search, images and advanced search.
So the base for this project was this form where you write a anything and push the boton "Google Search". The base of the search are quote items with type hidden, however the most important wos the item q.
